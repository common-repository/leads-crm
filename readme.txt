Add business customer leads management and customer relationship management (CRM) functionality for WordPress and WooCommerce

== Description ==

Enable Business Customer Relationship Management (CRM) tools for the WordPress WooCommerce platform. Easily prospect business using search tools and create lead business user accounts to track and manage the end to end sales process for customer acquisition.

= Main features: =
* Add a business lead for tracking the customer acquisition process
* Use the prospect tool for searching: Business Name, Business Type, Phone, Website and Address
* Bookmark businesses for personal follow-up and share bookmarks across the organization
* Provide an audit trail for managing the end to end sales process for a customer

== Installation ==

1. Upload Leads CRM WordPress WooCommerce to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now select Leads from the menu
4. If you have any issues contact. https://bizswoop.com/support/
